# Olist Brazilian E-Commerce Data Analytics Project
Analyzing Brazilian E-Commerce Public Dataset by Olist Store using Python, and Power BI.

##  Project Overview
End-to-end data analytics project analyzing 100K+ e-commerce orders to uncover
sales trends, customer behavior, and delivery performance.The analysis highlights 
revenue concentration, logistics impact on customer satisfaction, and opportunities for improving 
repeat purchases.

##  Key Business Questions
- How does revenue trend over time?
- Which product categories generate the most revenue?
- What is the customer retention rate?
- How does delivery delay affect customer reviews?
- What state contribute most to sales?


##  Key KPIs
- Total Revenue
- Total Orders & Total Customers
- Average Order Value (AOV)
- Retention & Churn Rate

##  Dashboard
Interactive Power BI dashboard with filters for date, category, and state.
- Monthly Revenue Trend 
- Top product Categories by Revenue
- Customer Lifetime Value (CLV)
- Retention & Churn Rate
